HTML 이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Web browsers received HTML documents from a web server or from local storage and render them into multimedia web pages.

HTML describes the structure of elements are building blocks of HTML pages. With HTML consturctes, images and other objects, such as interactive forms, may be embeded into the rendered page. It provides a means to create structured documents by denoting structureal semantics for text such as headings, paragraphs, lists, links, quotes, and other items. HTML elments are delineated by tags, written using angle brackets.
